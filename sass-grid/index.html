Exploration in a better kind of grid.

Goal: To create a semantic, flexible grid that also supports the golden ratio and modular scale.

Grid definition in the markup is optional - the formulae are both mixins and functions that calculate widths and margins based on pre-defined variables, so you can add them to any element, or define a class that you apply to your elements to make a more traditional grid.

Inspired by Semantic.gs and Bourbon, along with my distaste for html bloat in frameworks like bootstrap and 960.
