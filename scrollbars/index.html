Fun With Scrollbars

I wanted to understand how Google created and used their scrollbars in apps like Gmail.

This experiment is a result of replicating that behavior and style with just CSS. It achieves the following:

  • Styles the scrollbars using 
    -webkit-scrollbar
  • Is simply ignored in browsers that don't support it. (graceful fallback)
  • In some cases, they use a scrollbar that's visible on hover only. This is controlled by a class, which can be applied by javascript.

The CSS:

        /* A few base styles for presentation */
        .scroll {
          display: inline-block;
          vertical-align: top;
          overflow: hidden;
          margin-right:  10px;
          padding: 6px;
          border: 1px solid rgba(0, 0, 0, 0.2);
        }

        /* important parts: max-height and overflow-y */
        .scroll.one {
          max-height: 300px;
          width: 500px;
          overflow-y: scroll;
        }

        .scroll.two {
          max-height: 300px;
          width: 150px;
        }

        /*
          To prevent shifting of elements or
          accidental hiding of text when the scroll
          is shown, the contents must be wrapped in an
          element that can posess a margin.
        */
        .scroll.two > div {
          margin-right: 10px;
        }

        /* change the overflow state on hover */
        .scroll.two:hover {
          overflow-y: scroll;
        }

        .scroll::-webkit-scrollbar {
          width: 12px;
        }

        /* track: background of scroll area */
        .scroll::-webkit-scrollbar-track {
          background: transparent;
        }

        .scroll::-webkit-scrollbar-track:hover {
          background: transparent;
          border: 1 solid rgba(0, 0, 0, 0.2);
          border-top: 0px;
          border-bottom: 0px;
        }

        /* thumb: the "box" that moves when you scroll */
        .scroll::-webkit-scrollbar-thumb {
          background-color: rgba(0, 0, 0, 0.2);
        }

        .scroll:hover::-webkit-scrollbar-thumb {
          background-color: rgba(0, 0, 0, 0.3);
        }
